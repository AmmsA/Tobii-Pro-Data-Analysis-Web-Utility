Common Eye Tracking Measure Definitions

Base Measures
Metric Definition
Fixation Moment in time when the eye is relatively stationary.
Saccade Moments occurring between fixations, where the eye travels from one fixation to the next.
Scanpath A complete Saccade-Fixate-Saccade sequence.
Convex Hull Area The minimum spanning polygon for a set of points in a Cartesian coordinate system.
Pupil The hole located in the center of the iris of the eye.
Relative Angle The amount of rotation, in degrees between a Saccade and the horizontal axis.
Absolute Angle The amount of rotation, in degrees, from from the previous Saccade to the next Saccade.

Export Analysis Output Definitions

Measures of Search
Metric Definition
Fixation Count The number of discrete fixations captured during the experiment.
Saccade Count The number of discrete fixations captured during the experiment.
Average Saccade Length A central measure of pixel distance between fixations calculated from the Scanpath Length and divided by the Saccade Count.
Scanpath Length The sum of all Saccade Lengths denoting the total distance travelled by the participant's gaze over the course of the experiment.
Convex Hull Area The size, in pixels, of the region encapsulated by the user's Fixations in the Scanpath.
Measures of Processing
Metric Definition
Average Fixation Duration A central measure of the time a participant's gaze has remained on a single point.
Average Saccade Duration A central measure of the time a participant's gaze traveled between two Fixation points.
Saccade-to-Fixation Ratio The factor relating the total amount of time a participant spent in a Fixation gaze against a Saccade Gaze.
Measures of Cognitive Workload
Metric Definition
Average Left Pupil Dilation A central measure, in millimeters, of the diameter of a participant's left pupil over the course of an experiment.
Average Right Pupil Dilation A central measure, in millimeters, of the diameter of a participant's right pupil over the course of an experiment.
Sum of Absolute Saccadic Angles The total combined measure of the offset, in degrees, between the horizontal axis and the current fixation location where the previous fixation location is set as the origin.
Sum of Relative Saccadic Angles The total combined difference between in degrees traveled between the current and previous saccade where the current saccade is between the current and previous fixation.
Other Export Measures
Metadata
Metric Definition
Validity The ratio of valid to invalid samples in the export data, where any samples that did not have a validity code 0 for both the left and right eyes is considered invalid.
Duration The human readable time period elapsed from experiment start to finish
Counts
Metric Definition
Fixation Count The number of discrete gazes marked as Fixation during the experiment.
Saccade Count The number of discrete gazes marked as Saccade during the experiment.
Descriptive Statistics
Metric Definition
Mean The arithmetic mean found by dividing the set of values by the number of values in the set.
Median The middle value of the data set.
Mode The value in the data set which occurs most.
Variance Measure how how far the set of values spread from their mean.
Standard Deviation A measure of how much a variation exist in a data set, higher standard deviations denote a widely varying data set, while lower standard deviations denote a more constricted range of values.
Minimum The absolute smallest value within a data set.
Maximum The absolute largest value within a data set.
Sum The result of aggregating the data set through addition, giving total combined value of all entries in the set.
